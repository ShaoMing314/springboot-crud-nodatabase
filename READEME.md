# springboot入门crud
## 相关依赖
+ web
+ swagger
+ 增删改查
+ 没有数据库
+ 测试 crud 入门项目


没有依赖 , 直接运行
## 使用
+ 打包
```shell script
mvn clean package
```

+ 运行
```shell script
java -jar demo.jar
```

## swagger 访问网址
`localhost:8080/doc.html`
